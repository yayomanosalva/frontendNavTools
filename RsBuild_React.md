# Guía de instalación para rsbuild y React

[RsBuild - Guía de instalación para Rspack y React](https://rsbuild.dev/)

## 1. Instalar dependencias principales
```bash
bun add react-server-dom-webpack react-router-dom react-dom dotenv axios zod zustand @tanstack/react-query @tanstack/react-query-devtools clsx
```

```bash
bun add @radix-ui/react-dialog
```

## 2. Instalar dependencias de desarrollo
```bash
bun add -D @types/react @types/react-dom bun-types sass
```

## 3. Instalar dependencias de TailwindCss

[TailwindCSS - Guía de instalación para Rspack y React](https://tailwindcss.com/docs/installation/framework-guides/rspack/react)


```bash
bun add tailwindcss @tailwindcss/postcss postcss postcss-loader
```

-*rspack.config.ts*

```typescript
export default defineConfig({
  // ...
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ["postcss-loader"],
        type: "css",
      },
      // ...
    ],
  },
}
```

-*postcss.config.mjs*

```typescript
export default {
  plugins: {
    "@tailwindcss/postcss": {},
  },
};
```

-*index.css*
```css
@import "tailwindcss";
```

```bash
npm run dev
```

-*App.jsx*
```typescript
export default function App() {
  return (
    <h1 className="text-3xl font-bold underline">
      Hello world!
    </h1>
  )
}
```

*lib/utils.ts*
```typescript
import { clsx, type ClassValue } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```
